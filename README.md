# 个人简历网站

这是一个可部署到 GitHub Pages 的静态个人简历网站，包含：

- 关于我
- 实习经历
- 作品集

## 后续编辑

- 修改个人信息：编辑 `index.html` 顶部英雄区和联系信息。
- 新增作品：复制 `index.html` 中的 `<article class="work-card">...</article>`，粘贴到作品集区域的 `.work-grid` 内。
- 调整颜色：编辑 `styles.css` 顶部的 `:root` 变量。

## GitHub Pages

仓库推送到 GitHub 后，工作流会自动发布站点。也可以在 GitHub 仓库的 Settings → Pages 中选择 GitHub Actions 作为部署来源。
